# Jimmy, Jianing, JIA

Hi, I'm Jimmy. I'm a software engineer, an avigeek, and a photographer.

Theoretically, it looks more professional if the so called personal blog can be filled with skill sets, which is way more attractive for ppl to quickly understand a more software engineer side of me. However, my intuition told me no.

Doing useless things is always interesting. As a result, I've build this website to function as a gallery to host all of the plane spottings I had. I've started doing this from August 2025, and I'm still trying to stick to it, which is probably my only interest.

**You are greatly welcomed.** I'm more than just appreciate to see ppl get attracted by something that I've built up. Hope that you might understand more about my pale life so that next time, when you can't get in touch with me on our spare time, you'll take a better guess on what I'm doing.

The whole webapp is built based on Spring-boot, React, and cups of tea.
